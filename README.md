# Repo for testing candidate's skills

This repository contains list of tests for candidate skills:

- [01_SQL](01_SQL) - test basic SQL skills.
- [02_Python](02_Python) - test basic Python skills.
- [03_Docker](03_Docker) - test basic Docker and Bash skills.

## Notes for BI

- Clone this repo.
- On your Github create new private repo e.g. candidate-skills-<candidate_name>.
- Push this repository to Github.
- Ask candidate for Github account name and share Github repo with him.
- Give a candidate some instructions.
